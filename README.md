# CasinoProject

To run the project, download the repository’s source code by pressing the green code button and downloading it as a zip file. Extract the contents of the zip file and run the .slnx file in the folder. **THIS BUILD IS ONLY FOR Visual Studio 2022 AND WILL NOT WORK ON OTHER VERSIONS.** Open the Solution Explorer if it is collapsed on the right and open the main.c file. Run the program as normal by clicking the green run button at the top of the program.
